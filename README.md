# Time-Series-Data-Analysis-and-Trend-Discovery-in-Pakistan-Crop-Prices

## 📌 Overview

This project presents a complete data mining pipeline that combines machine learning, time series analysis, and clustering techniques to analyze and model complex datasets. The objective is to extract meaningful insights, identify patterns, and evaluate predictive performance using multiple analytical approaches.

---

## 🎯 Objectives

* Perform data preprocessing and cleaning
* Apply feature engineering techniques
* Build and evaluate machine learning models
* Conduct time series analysis (ACF, PACF, forecasting)
* Perform clustering to identify hidden patterns
* Save and analyze model outputs and evaluation metrics

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib
* Statsmodels

## 📊 Methodology

### 1. Data Preprocessing

* Handling missing values
* Data cleaning and transformation

### 2. Feature Engineering

* Creation of lag features
* Time-based feature extraction

### 3. Machine Learning Models

* Logistic Regression
* Random Forest
* Model evaluation using:

  * Accuracy
  * Precision
  * Recall
  * F1-score

### 4. Time Series Analysis

* ACF and PACF plots
* Trend and seasonality analysis
* Forecasting using statistical models

### 5. Clustering

* Identification of patterns using clustering algorithms
* Visualization of clusters
